Política de Privacidade

Esta política descreve como o Pro'B coleta, usa e protege os dados pessoais dos usuários da plataforma.

1. Dados coletados

Podemos coletar nome, telefone, e-mail e informações de agendamento necessárias para o funcionamento do serviço.

2. Uso dos dados

Os dados são utilizados exclusivamente para viabilizar o agendamento, comunicação entre cliente e barbearia, e melhoria da plataforma.

3. Compartilhamento

Não compartilhamos dados pessoais com terceiros, exceto quando exigido por lei ou necessário para o funcionamento técnico do serviço (ex.: provedores de infraestrutura).

4. Segurança

Adotamos medidas técnicas razoáveis para proteger os dados armazenados.

5. Direitos do usuário

Você pode solicitar a correção ou exclusão dos seus dados a qualquer momento, em conformidade com a LGPD.
